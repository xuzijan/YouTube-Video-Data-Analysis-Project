🏗️ 技术栈
  • Python 3.x
  • 数据处理: pandas, numpy
  • 机器学习: scikit-learn, xgboost
  • 可视化: matplotlib, seaborn

🎯 算法运用
  分类算法:
    • 逻辑回归 (Logistic Regression)
    • 随机森林 (Random Forest Classifier)
    • 支持向量机 (SVM)
    
  聚类算法:
    • K-means聚类
    • 肘部法则 (Elbow Method)
    • 轮廓系数分析 (Silhouette Analysis)
    
  回归算法:
    • 随机森林回归 (Random Forest Regressor)
    • XGBoost回归
    
  特征工程:
    • 多维特征构造
    • 标准化处理 (StandardScaler)
    • 特征相关性分析
    • 特征重要性排序

  • 数据规模: 155,669条YouTube视频记录
  • 时间跨度: 2006年11月 - 2025年10月 (19年)
  • 特征扩展: 9个原始特征 → 59个工程特征
  • 特征分类: 文本(12) + 时间(17) + 参与(19) + 频道(15) + 复合(8)
  • 处理流程: 数据预处理 → 特征工程 → EDA分析 → 机器学习建模 → 可视化呈现

项目核心成果
════════════════════════════════════════════════════════════════════════════════════════════════════════

📊 数据分析发现
  
  1️⃣ 顶级视频特征分析
     • 观看数最高: 数十亿级别播放量
     • 点赞数最高: 千万级别互动
     • 评论数最高: 百万级别讨论
     • 参与率分布: 呈现长尾分布特征
  
  2️⃣ 时间模式分析
     • 发布时间分布: 识别最佳发布时段
     • 周期性规律: 星期几、月份的影响
     • 时段对比: 黄金时段 vs 普通时段
     • 发布趋势: 19年历史数据的演变
  
  3️⃣ 频道对比分析
     • 频道规模差异: 大小频道的显著差距
     • 一致性分析: 频道内容稳定性评估
     • 等级分类: 小型、中型、大型、超大型频道
     • 性能对比: 不同规模频道的表现差异
  
  4️⃣ 参与度深度分析
     • 参与率分布: 整体参与率统计特征
     • 病毒视频识别: 高参与度视频的特征
     • 互动模式: 点赞与评论的比例关系
     • 分段分析: 不同观看量级别的参与度差异

🤖 机器学习模型
  
  分类模型 (视频参与度预测):
     • 逻辑回归: 基准模型
     • 随机森林: 最优性能
     • 支持向量机: 高精度分类
     • 评估指标: 准确率、精确率、召回率、F1分数
  
  聚类模型 (频道类型分群):
     • K-means算法: 频道自动分类
     • 最优K值: 通过肘部法则和轮廓系数确定
     • 聚类特征: 基于频道规模和参与度
     • 应用价值: 差异化运营策略制定
  
  预测模型 (性能预测):
     • 随机森林回归: 主力模型
     • XGBoost回归: 增强预测
     • 评估指标: R²、RMSE、MAE
     • 特征重要性: 识别关键影响因素

📈 特征工程成果
  
  特征类别分布:
     • 文本特征 (12个): 标题长度、词数、特殊符号、大写比例等
     • 时间特征 (17个): 年份、月份、星期、小时、时段、节日等
     • 参与特征 (19个): 参与率、点赞率、评论率、病毒标识等
     • 频道特征 (15个): 频道平均值、总计、排名、等级、一致性等
     • 复合特征 (8个): 视频与频道对比、内容质量评分等
  
  特征重要性:
     • Top特征: 通过随机森林识别最关键特征
     • 相关性分析: 特征间的相关性热力图
     • 方差分析: 特征信息量评估
